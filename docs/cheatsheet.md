# Keyboard cheat sheet

A one-page reference you can print or keep beside your terminal.

## Movement

| Key | Action |
|-----|--------|
| `â†‘` / `k` | move selection up |
| `â†“` / `j` | move selection down |

## Branch actions

| Key | Action |
|-----|--------|
| `Enter` / `c` | checkout selected branch |
| `d` | delete branch (with confirmation) |
| `s` | sync with trunk |
| `r` | restack |
| `S` | submit stack |

## PRs

| Key | Action |
|-----|--------|
| `o` / `O` | open PR / stack page on Graphite |
| `g` | open PR on GitHub |
| `y` | copy PR url or branch name |

## Misc

| Key | Action |
|-----|--------|
| `/` | fuzzy filter |
| `R` | refresh |
| `?` | help |
| `q` | quit |
